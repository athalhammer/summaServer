SUMMARUM+


Overview

How to use the SUMMARUM+ service:

  • query parameter entity*: the URI of the DBpedia resource
  • query parameter topK*: the number of triples the summary should contain
  • query parameter maxHops the maximum number of hops in the graph the result should contain.
  • query parameter language defines the language of the output literals. We recommend to use RFC 4646.
  • query parameter fixedProperty: the URI of a DBpedia predicate. If this parameter is present, the
    system focuses on statements that involve the given entity in combination with the given predicate.
The parameters marked with the star symbol (*) are mandatory.

Example queries:

HTML

summarum?entity=http://dbpedia.org/resource/Vladimir_Putin&topK=5
summarum?entity=http://dbpedia.org/resource/Original_Sin_(2001_film)&topK=5
summarum?entity=http://dbpedia.org/resource/Usain_Bolt&topK=5
summarum?entity=http://dbpedia.org/resource/Marie_Curie&topK=10
summarum?entity=http://dbpedia.org/resource/Manchester_City_F.C.&topK=10
summarum?entity=http://dbpedia.org/resource/Lexus&topK=10
summarum?entity=http://dbpedia.org/resource/The_Cosby_Show&topK=10

Turtle (to copy&paste for console) 
curl -i -d "[ a <http://purl.org/voc/summa/Summary> ; \
<http://purl.org/voc/summa/entity> \
<http://dbpedia.org/resource/Porsche> ; \
<http://purl.org/voc/summa/topK> '5'; \
<http://purl.org/voc/summa/maxHops> '1'; \
<http://purl.org/voc/summa/language> 'en'; \
<http://purl.org/voc/summa/fixedProperty> \
<http://dbpedia.org/ontology/keyPerson>; \
<http://purl.org/voc/summa/fixedProperty> \
<http://dbpedia.org/ontology/owner>  ] ." \
-H "content-type: text/turtle" -H "accept: text/turtle" http://km.aifb.kit.edu/summaPlus/summarum
